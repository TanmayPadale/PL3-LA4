  1. License-Plate Detection

    Domain- Digital Image Processing,Machine Learning

    Technology Stack- Python,OpenCV

    Abstract-

    Automatic license plate recognition is a computer vision technology to extract license number of vehicles from image. Security is big issue in highly restricted areas as well as in public places.
    The approach used to segment the images is Connected Component Analysis. Connected regions will imply that all the connected pixels belong to the same object. A pixel is said to be connected to another if they both have the same value and are adjacent to each other.

  2. Fundraiser Managesment using Blockchain

    Domain- Blockchain, cryptography

    Technology Stack-Python Front end using HTML ,CSS & JS, Node JS, Interface, Web, Meta mask extension for browsing

    Abstract-

    Ethereum is an open source, public blockchain based distributed computing platform and operating system featuring smart contract functionality. Ether is a cryptocurrency generated by the Ethereum platform and used to compensate mining nodes for computations performed. Each Ethereum account has an ether balance and ether may be transferred from one account to another. The purpose of Fundraising Management system using block chain is to bring transparency in money handling ways of charitable organizations and to develop trust among the general mass in web based charity . As the transactions performed by the charity organization will be refined and displayed on our website, thus it will be helpful for the donors and organization to build trust among themselves.
